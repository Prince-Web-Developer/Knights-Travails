# Knights Travails

a program which tells the knight moves in the shortest path from one place to another on a chess board

## 📥 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

List the software, languages, or tools required to run this project:

- Node.js
- npm

### Installation

1. Clone the repository:
2. Navigate to the project directory:
   ```bash
   cd Knights-Travails
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
